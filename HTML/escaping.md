7/26/17 Escaping Ampersands in HTML entities

To escape an ampersand so that it shows up in text, use 
`&amp;`


*Example:* 


Write     | Result    | 
----------|-----------|
PB\&amp;J | PB&J      | 



 <br />


To escape a left angle bracket, use `&lt;`


*Example:*



Write     | Result    | 
----------|-----------|
3 \&lt; 8 | 3 < 8     |
