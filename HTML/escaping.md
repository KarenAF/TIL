7/26/17 Escaping Ampersands in HTML entities

To escape an ampersand so that it shows up in text, use 
`&amp;`


*Example:* 
<table>
    <tr>
        <td>PB\&amp;J</td>
        <td>PB&J</td>
    </tr>
</table>

Write     | Result    | 
----------|-----------|
PB\&amp;J | PB&J      | 



 <br />


To escape a left angle bracket, use `&lt;`


*Example:*

<table>
    <tr>
        <td> 3 \&lt; 8 </td>
        <td> 3 < 8 </td>
    </tr>
</table>

Write     | Result    | 
----------|-----------|
3 \&lt; 8 | 3 < 8     |
